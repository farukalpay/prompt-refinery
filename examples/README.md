# Example Artifacts

This folder contains full example payloads referenced from the main README.

- `soc2_input_prompt.txt`: CLI input prompt text
- `soc2_quality_targets.txt`: quality targets used for the SOC2 example
- `soc2_repaired_prompt.txt`: full repaired prompt output shown in README
- `launch_checklist_payload.json`: full JSON output example for `--json`
- `mcp_tools_call_request.json`: MCP `tools/call` request example
- `mcp_tools_call_response.json`: MCP `tools/call` response example
